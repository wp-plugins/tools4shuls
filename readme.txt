A plugin that provides interface for working with the Tools4Shuls system directly within a client's WordPress installation.

== Description ==

By installing this plugin, a user of Tools4Shuls can work with their account through their WordPress dashboard. *Note that not all Tools4Shuls modules are currently supported by the plugin. Currently supported modules: Calendar and Donations.

== Installation ==

1. Upload the entire t4s directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Work with the plugin via the Tools4Shuls menu section of the dashboard.
